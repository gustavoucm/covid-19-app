# Detalles técnicos

## Instalación

```bash
npm install
```

### Para ejecutar en modo de desarrollo de manera local
```bash
npm run serve
```

### Para crear el archivo compilado para producción
```bash
npm run build
```

### Para ejecutar lints
```bash
npm run lint
```
